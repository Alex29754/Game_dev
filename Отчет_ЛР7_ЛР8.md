## 2. Улучшение управления персонажем

Для улучшения ощущения управления были добавлены ускорение, плавное изменение скорости и прыжок.

Теперь персонаж может:

* двигаться по сцене;
* ускоряться при зажатии клавиши `Shift`;
* прыгать при нажатии клавиши `Space`;
* плавно разгоняться и останавливаться;
* поворачиваться в сторону движения.

Фрагмент кода из `LabPlayerController.cs`:

```csharp
public float speed = 5.5f;
public float sprintSpeed = 8f;
public float acceleration = 22f;
public float deceleration = 28f;
public float jumpForce = 6.5f;

private Vector3 movement;
private Vector3 currentVelocity;

private bool jumpQueued;
private bool isGrounded;
```

Обработка прыжка:

```csharp
if (Input.GetKeyDown(KeyCode.Space))
{
    jumpQueued = true;
}

if (jumpQueued && isGrounded)
{
    rb.linearVelocity = new Vector3(rb.linearVelocity.x, 0f, rb.linearVelocity.z);
    rb.AddForce(Vector3.up * jumpForce, ForceMode.VelocityChange);
    isGrounded = false;
}
```

Для проверки нахождения персонажа на земле используется `SphereCastAll`. Это нужно, чтобы прыжок срабатывал только тогда, когда игрок действительно стоит на платформе.

    ParticleSystem particles = particleObject.AddComponent<ParticleSystem>();
    ParticleSystem.MainModule main = particles.main;

## 7. Добавление пушек и выстрелов

Для повышения динамики были добавлены пушки.

Пушки:

* находятся на уровне;
* поворачиваются в сторону игрока;
* периодически стреляют ядрами;
* наносят урон при попадании;
* сопровождаются звуком и эффектом частиц.

Фрагмент кода из `LabInteractables.cs`:

```csharp
public class LabCannonShooter : MonoBehaviour
{
    public float fireInterval = 2.4f;
    public float projectileSpeed = 8.5f;
    public float fireRange = 18f;
    public int damage = 1;

    private LabGameManager manager;
    private Transform target;

    private void Update()
    {
        fireTimer -= Time.deltaTime;

        if (fireTimer > 0f)
        {
            return;
        }

        Vector3 muzzlePosition = transform.position + Vector3.up * 0.65f;
        Vector3 aimPosition = target.position + Vector3.up * 0.6f;
        Vector3 toTarget = aimPosition - muzzlePosition;

        if (toTarget.magnitude <= fireRange)
        {
            Fire(muzzlePosition, toTarget.normalized);
        }

        fireTimer = fireInterval;
    }
}
```

Логика попадания ядра:

```csharp
private void OnTriggerEnter(Collider other)
{
    if (other.GetComponentInParent<LabPlayerController>() != null)
    {
        manager.TakeDamage(damage, "Попадание ядра: здоровье уменьшилось.");
        Destroy(gameObject);
    }
}
```

## 4. Логика победы

Победа наступает только при выполнении условий:

* собраны все 3 ключа;
* дверь открыта;
* игрок дошёл до финишной зоны.

Фрагмент кода:

```csharp
public void TryFinish()
{
    if (!doorOpened || collectedItems < TotalItems)
    {
        ShowHint("Финиш закрыт: нужны все ключи и открытая дверь.", 3f);
        return;
    }

    state = GameState.Victory;
    Time.timeScale = 0f;
    player.InputEnabled = false;
    overlayPanel.SetActive(true);
    overlayText.text = "Победа!";
}
```

После победы появляется сообщение на экране, игра останавливается, а игрок может нажать `R` для рестарта.

## 5. Логика поражения

Поражение наступает, если здоровье игрока становится равным нулю.

Здоровье уменьшается при:

* попадании в опасную зону;
* попадании ядра пушки;
* падении за пределы уровня.

Фрагмент кода:

```csharp
public void TakeDamage(int damage, string reason)
{
    if (state != GameState.Playing || damageCooldown > 0f)
    {
        return;
    }

    health -= damage;

    if (health <= 0)
    {
        health = 0;
        state = GameState.Defeat;
        Time.timeScale = 0f;
        player.InputEnabled = false;
        overlayPanel.SetActive(true);
    }
}
```

При поражении показывается отдельное сообщение и предлагается перезапустить уровень.
